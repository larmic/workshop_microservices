# Microservices Workshop

Educational workshop project (German language) teaching microservice architecture patterns and best practices.

## Tech Stack

- Java with GraalVM 25
- Build system: not yet configured

## Workshop Topics

The workshop covers microservice fundamentals and patterns including:

- Twelve-Factor App methodology
- Health Checks
- Circuit Breaker pattern
- API Gateway
- Service Discovery
- Saga Pattern
- Event Sourcing
- CQRS
- Downtimeless Deployment

Example domain: Travel booking system (Hotel, Flight, Car rental)
